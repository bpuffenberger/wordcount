# Word Count Exclusion Tool

This is a React + Vite app configured to deploy to GitHub Pages using GitHub Actions.

## Repository name
wordcount

## Deployment
This project is configured for GitHub Pages at:

https://YOUR-USERNAME.github.io/wordcount/

## Required GitHub Pages setting
In GitHub:

- Go to **Settings > Pages**
- Under **Build and deployment**
- Set **Source** to **GitHub Actions**

## Local development
```bash
npm install
npm run dev
